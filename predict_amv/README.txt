README

This folder contains the code used for training/analysis/visualization of the results.


Directories ---

    Analysis      : Visualization Code
    Figures       : Visualization code outputs here
    Data          : Move results into the metrics folder for visualization
    Models        : Scripts for training/testing models and generating results
    Preprocessing : Script for preprocessing/regridding data


Note --

In the same directory which contains predict_amv, place the CESM_data folder.
